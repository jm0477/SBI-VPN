接下来读读新的vpn使用说明吧：
1.启动vpn：
	首先我们需要打开桌面上的proxy图标（就像原来打开GoAgent一样），然后最小化，不要理他 
	ps: 千万不要关闭"Start_VPN"，关闭了就不能翻墙了。

	如果你不需要翻墙，可以点击桌面上的Stop-VPN图标来结束vpn
	ps：如果还是不能上网，请重新启动chrome浏览器

新的vpn系统，会让整个系统默认翻墙，所以无需在安装任何插件，可以卸载chrome的所有插件。
！！！最最重要的是， 运行脚本之前，一定要确保各种杀毒软件处于关闭状态，否则会出现各种拦截导致vpn不可用，排查问题很困难。
